## <a name="create-an-iot-hub"></a>Создание Центра Интернета вещей
Создайте центр Интернета вещей, к которому будет подключено приложение, имитирующее устройство. Ниже показано, как это сделать с помощью портала Azure.

1. Войдите на [портал Azure][lnk-portal].

1. Выберите **Создать** > **Интернет вещей** > **Центр Интернета вещей**.
   
    ![Навигационная панель портала Azure][1]

1. В области **Центр Интернета вещей** введите следующие сведения для создания Центра Интернета вещей:

   * **Имя**.Создайте имя Центра Интернета вещей. Если введенное имя является допустимым, появится зеленая галочка.

   [!INCLUDE [iot-hub-pii-note-naming-hub](iot-hub-pii-note-naming-hub.md)]

   * **Ценовая категория и категория масштабирования**. Для этого руководства выберите **бесплатную категорию F1**. Дополнительные сведения о ценовых категориях и категориях масштабирования см. в [этой статье][lnk-pricing].

   * **Группа ресурсов**. Создайте группу ресурсов для размещения Центра Интернета вещей или используйте имеющуюся. Дополнительные сведения см. в статье [Управление ресурсами Azure через портал][lnk-resource-groups]

   * **Расположение**: выберите ближайшее расположение.

   * **Закрепить на панели мониторинга**. Установите этот флажок, чтобы быстро открывать Центр Интернета вещей с помощью панели мониторинга.

    ![Окно "Центр IoT"][2]

1. Нажмите кнопку **Создать**. Создание Центра Интернета вещей может занять несколько минут. Ход создания можно отслеживать на панели **уведомлений**.

1. Когда центр Интернета вещей будет создан, щелкните соответствующую плитку на портале Azure и откройте окно свойств. После создания Центра Интернета вещей найдите важные данные для подключения к нему устройств и приложений. Запишите **имя узла**, а затем щелкните **Политики общего доступа**.
   
    ![Новое окно центра Интернета вещей][4]

1. В колонке **Политики общего доступа** выберите политику **iothubowner**, а затем запишите строку подключения в окне **iothubowner**. Дополнительные сведения см. в разделе [Контроль доступа][lnk-access-control] в руководстве разработчика для Центра Интернета вещей Azure.
   
    ![Политики общего доступа][5]
