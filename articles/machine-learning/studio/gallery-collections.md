# <a name="discover-and-share-collections-in-azure-ai-gallery"></a>Поиск и публикация коллекций в коллекции решений ИИ Azure
[!INCLUDE [machine-learning-gallery-item-selector](../../../includes/machine-learning-gallery-item-selector.md)]

## <a name="collections"></a>Коллекции
[Коллекция](https://gallery.cortanaintelligence.com/collections) позволяет объединить эксперименты, интерфейсы API и другие элементы, относящиеся к конкретному решению или понятию. Их можно объединять в группы для дальнейшего использования или предоставления к ним общего доступа.

## <a name="discover"></a>Поиск
Чтобы перейти к [коллекциям](http://gallery.cortanaintelligence.com), щелкните **More** (Дополнительно) и выберите **Collections** (Коллекции).

![Выбор коллекций на начальной странице коллекции](./media/gallery-collections/select-collections-in-gallery.png)

На странице **[Collections](https://gallery.cortanaintelligence.com/collections)** (Коллекции) отображен список недавно добавленных и популярных коллекций. Выберите элемент **See all** (Просмотреть все), чтобы просмотреть все коллекции. Чтобы найти нужную коллекцию, выберите **See all** (Просмотреть все), а затем задайте условия фильтра. Вы можете также ввести условия поиска в поле **поиска** в верхней части страницы коллекции.

Подробные сведения о коллекции можно получить на странице дополнительных сведений о ней. Чтобы открыть страницу сведений о коллекции, выберите коллекцию. На этой странице в разделе **Comments** (Комментарии) можно оставлять комментарии, отзывы и задавать вопросы о коллекции. Можно даже поделиться коллекцией с друзьями и коллегами на Twitter и LinkedIn. Кроме того, можно отправить ссылку на страницу сведений о коллекции по электронной почте, чтобы другие пользователи также могли просмотреть эту страницу.

![Расскажите друзьям о том, что вы нашли](./media/gallery-how-to-use-contribute-publish/share-links.png)

![Добавляйте свои комментарии](./media/gallery-how-to-use-contribute-publish/comments.png)

## <a name="contribute"></a>Участие
Можно создать коллекцию, которая будет содержать собственные элементы или элементы, созданные сообществом. Элементами коллекции могут быть любые элементы, касающиеся определенного решения или понятия.

Например, в коллекции можно объединить элементы по определенной теме или элементы многоэтапного эксперимента, который решает сложные задачи. Изначальные коллекции, добавленные корпорацией Майкрософт, представляют собой шаблоны многоэтапных экспериментов машинного обучения. Эти шаблоны можно использовать для решения реальных проблем, таких как обнаружение онлайн-мошенничества, классификация текста, прогнозирование в розничной торговле и прогнозирование обслуживания.

### <a name="create-a-collection"></a>Создание коллекции

1. Войдите в коллекцию с помощью учетной записи Майкрософт.

2.  Выберите свое изображение в верхней части страницы, а затем выберите свое имя.
  
    ![Выбор имени своей учетной записи](./media/gallery-collections/click-account-name.png)

3. Выберите **New Item** (Создать элемент).
   
    ![Выбор пункта New Item (Создать элемент)](./media/gallery-collections/click-new-item.png)
4. На странице **описания** в раскрывающемся списке **Item type** (Тип элемента) выберите **Collection** (Коллекция). Введите имя для коллекции, краткий обзор, подробное описание и любые теги, которые могут помочь другим пользователям найти вашу коллекцию.
   
    ![Указание сведений о новой коллекции](./media/gallery-collections/create-collection-page-1.png)
5. На странице **Image Selection** (Выбор изображения) выберите изображение, которое будет отображаться в коллекции. Можно отправить свой файл изображения или выбрать готовое изображение. Выберите изображение, которое может помочь пользователям определить содержимое и назначение коллекции.
   
    ![Указание сведений о новой коллекции](./media/gallery-collections/create-collection-page-2.png)
6. На странице **параметров** для параметра **Visibility** (Видимость) выберите для коллекции значение **Public** (Общедоступная) (любой пользователь сможет просмотреть коллекцию) или **Unlisted** (Вне списка) (просмотреть коллекцию смогут только люди, которым предоставлена прямая ссылка).
   
   > [!IMPORTANT]
   > После того как вы сделаете коллекцию **общедоступной**, а затем выберите **Создать**, параметр **Unlisted** (Вне списка) станет недоступен для коллекции.
   > 
   > 
   
    ![Выбор Public (Общедоступная) или Unlisted (Вне списка)](./media/gallery-collections/create-collection-page-3.png)
7. Нажмите кнопку **Создать**.

Эта коллекция теперь доступна в коллекции решений ИИ Azure. Ваши коллекции появятся на вкладке **Collection** (Коллекция) на странице вашей учетной записи.

### <a name="add-items-to-a-collection"></a>Добавление элементов в коллекцию
Чтобы добавить элементы в коллекцию, откройте коллекцию, выберите **Edit** (Изменить), а затем **Add Item** (Добавить элемент).

![Добавление элементов в коллекцию на странице сведений](./media/gallery-collections/add-to-collection-from-details-page.png)

На странице учетной записи отобразится список элементов, которые вы отправили в коллекцию. Вы также можете выполнить в коллекции поиск элементов для добавления. Выберите элементы, которые требуется добавить. Каждый выбранный элемент включается в набор элементов для добавления. На кнопке **Add** (Добавить) отображается количество выбранных элементов.

![Добавление в коллекцию](./media/gallery-collections/add-to-collection.png)

Если при просмотре коллекции вы нашли элемент, который хотите добавить, выберите **Add to collection** (Добавить в коллекцию), чтобы открыть элемент. Выберите коллекцию, в которую необходимо добавить элемент.

![Добавление элементов в коллекцию на странице сведений](./media/gallery-collections/add-to-collection-from-item-details.png)

Для изменения сводки, описания или тегов коллекции откройте ее и выберите команду **Edit** (Изменить). 

При редактировании коллекции в ней можно изменить порядок элементов. Используйте кнопки со стрелками рядом с элементом, чтобы переместить его в списке вверх или вниз. 

К элементам в коллекции можно добавить примечания. Для этого щелкните в правом верхнем углу элемента и выберите команду **Add/Edit note** (Добавить/изменить примечание). 

Чтобы удалить элемент из коллекции, выберите команду **Remove** (Удалить).

![Изменение сведений о коллекции](./media/gallery-collections/change-collection-details.png)
