# <a name="storsimple-data-manager-overview-private-preview"></a>Общие сведения о диспетчере данных StorSimple (закрытая предварительная версия)

## <a name="overview"></a>Обзор

Решение гибридного облачного хранилища Microsoft Azure StorSimple устраняет характерные для неструктурированных данных сложности, которые обычно возникают с файловыми ресурсами. StorSimple использует облачное хранилище в качестве расширения локального решения и автоматически связывает данные локального хранилища с данными облачного хранилища. Интегрированная защита данных, а также локальные и облачные моментальные снимки устраняют необходимость в расширении инфраструктуры хранилища. Архивирование и аварийное восстановление также легко выполняется. Облако при этом выступает в качестве удаленного расположения.

Служба преобразования данных, представленная в этом документе, упрощает доступ к данным StorSimple в облаке. Эта служба предоставляет API-интерфейсы для извлечения данных из StorSimple и их представления другим службам Azure в удобных форматах. В этой предварительной версии поддерживаются такие форматы, как большие двоичные объекты Azure и ресурсы служб мультимедиа Azure. Благодаря этому преобразованию можно легко подключить службы, например службы мультимедиа Azure, Azure HDInsight, Машинное обучение Azure и Поиск Azure, для работы с данными на локальном устройстве StorSimple серии 8000.

Ниже представлена общая блок-схема, иллюстрирующая этот процесс.

![Общая схема](./media//storsimple-data-manager-overview/high-level-diagram.png)

В этом документе объясняется, как можно зарегистрироваться для получения закрытой предварительной версии этой службы. Здесь также объясняется, как написать код приложений, использующих данные StorSimple и других служб Azure в облаке, с помощью этой службы.

## <a name="sign-up-for-data-manager-preview"></a>Регистрация для получения предварительной версии диспетчера данных
Прежде чем зарегистрироваться для получения службы диспетчера данных просмотрите следующие предварительные требования.

### <a name="prerequisites"></a>Предварительные требования

В этом упражнении предполагается, что у вас есть:
* активная подписка Azure;
* доступ к зарегистрированному устройству StorSimple серии 8000;
* все ключи, связанные с устройством StorSimple серии 8000.

### <a name="sign-up"></a>Регистрация

Диспетчер данных StorSimple доступен в качестве закрытой предварительной версии. Чтобы зарегистрироваться для получения закрытой предварительной версии этой службы, сделайте следующее:

1.  Войдите на портал Azure, используя расширение диспетчера данных StorSimple, расположенное по адресу [https://aka.ms/HybridDataManager](https://aka.ms/HybridDataManager). Войдите, используя учетные данные Azure.

2.  Щелкните значок **+** для создания службы. Щелкните **Хранилище** и в открывшейся вкладке щелкните **Показать все**.

    ![Поиск значка диспетчера данных StorSimple](./media/storsimple-data-manager-overview/search-data-manager-icon.png)

3. Отобразится значок диспетчера данных StorSimple.

    ![Выбор значка диспетчера данных StorSimple](./media/storsimple-data-manager-overview/select-data-manager-icon.png)

4. Щелкните значок диспетчера данных StorSimple и нажмите кнопку **Создать**. Выберите подписку, которую следует включить для закрытой предварительной версии, а затем нажмите кнопку **Sign me up!** (Зарегистрироваться).

    ![Кнопка "Зарегистрироваться"](./media/storsimple-data-manager-overview/sign-me-up.png)

5. После этого вам будет отправлен запрос на внедрение. Мы выполним эту операцию в кратчайшие сроки. После включения подписки можно создать службу диспетчера данных StorSimple.

6. Для простого доступа к службе диспетчера данных StorSimple щелкните значок со звездочкой, чтобы закрепить службу в избранном.

    ![Доступ к диспетчеру данных StorSimple](./media/storsimple-data-manager-overview/access-data-managers.png)
