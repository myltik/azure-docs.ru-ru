# <a name="use-the-storsimple-device-manager-service-to-manage-shares-on-the-storsimple-virtual-array"></a>Управление общими папками в виртуальном массиве StorSimple с помощью диспетчера устройств StorSimple

## <a name="overview"></a>Обзор

В этом руководстве объясняется, как использовать службу диспетчера устройств StorSimple для создания общих папок и управления ими в виртуальном массиве StorSimple.

Служба диспетчера устройств StorSimple — это расширение портала Azure, которое позволяет управлять решениями StorSimple с помощью единого веб-интерфейса. Кроме управления общими папками и томами, с помощью службы диспетчера устройств StorSimple можно просматривать устройства и управлять ими, просматривать оповещения, управлять политиками резервного копирования и каталогом резервных копий.

## <a name="share-types"></a>Типы общих папок

Есть два типа общих папок StorSimple.

* **Локально закрепленные** — данные в этих общих папках всегда остаются в массиве и не переносятся в облако.
* **Многоуровневые** — данные в этих общих папках могут переноситься в облако. При создании многоуровневой общей папки подготавливается приблизительно 10 % пространства на локальном уровне и 90 % пространства в облаке. Например, если подготовлена общая папка объемом 1 ТБ, 100 ГБ будут находиться в локальном пространстве, а 900 ГБ будут использоваться в облаке для распределения данных по уровням. Это означает, что, если на устройстве не хватает локального пространства, то нельзя подготовить многоуровневую общую папку (так как 10 % места на локальном уровне не будут доступны).

### <a name="provisioned-capacity"></a>Подготовленная емкость

В таблице ниже приведены значения максимальной подготовленной емкости для каждого типа общей папки.

| **Идентификатор ограничения** | **Ограничение** |
| --- | --- |
| Минимальный размер многоуровневого общего файлового ресурса |500 ГБ |
| Максимальный размер многоуровневого общего файлового ресурса |20 TБ |
| Минимальный размер локально закрепленного общего файлового ресурса |50 ГБ |
| Максимальный размер локально закрепленного общего файлового ресурса |2 ТБ |

## <a name="the-shares-blade"></a>Колонка "Общие папки"

Меню **Общие папки** в колонке сводки службы StorSimple отображает список общих папок хранилища в данном массиве StorSimple и позволяет управлять ими.

![Колонка "Общие папки"](./media/storsimple-virtual-array-manage-shares/shares-blade.png)

Общая папка обладает рядом атрибутов.

* **Имя общей папки** — уникальное описательное имя, которое помогает идентифицировать общую папку.
* **Состояние** — том может быть включен или отключен. Если общая папка находится в автономном режиме, пользователи не смогут получить к ней доступ.
* **Тип** — указывает тип общей папки: **Многоуровневая** (по умолчанию) или **Локально закрепленная**.
* **Емкость** — используемый объем данных по сравнению с общим объемом данных, который может быть сохранен в общей папке.
* **Описание** — необязательный параметр, который помогает описать общую папку.
* **Разрешения** — разрешения NTFS для общей папки, которыми можно управлять в проводнике.
* **Резервное копирование** — в виртуальном массиве StorSimple резервное копирование включено автоматически для всех общих папок.

![Сведения об общих папках](./media/storsimple-virtual-array-manage-shares/share-details.png)

В этом руководстве приведены инструкции по выполнению следующих задач:

* добавление общей папки;
* изменение общей папки;
* отключение общей папки;
* удаление общей папки.

## <a name="add-a-share"></a>Добавление общей папки

1. В колонке сводки службы StorSimple щелкните **+ Добавить общую папку** на панели команд. Откроется колонка **Добавить общую папку**.

    ![Добавление общей папки](./media/storsimple-virtual-array-manage-shares/add-share.png)

2. В колонке **Добавить общую папку** выполните следующие действия.
   
    1. В поле **Имя общей папки** введите уникальное имя. Оно должно представлять собой строку длиной от 3 до 127 символов.

    2. Необязательное **описание** общей папки. Оно поможет определить ее владельцев.

    3. В раскрывающемся списке **Тип** укажите, общую папку какого типа следует создать: **многоуровневую** или **локально закрепленную**. Для рабочих нагрузок, которые требуют локальных гарантий, низкой задержки и более высокой производительности, выберите тип **Локально закрепленная**. Для других данных выберите тип **Многоуровневая**.

    4. В поле **Емкость** укажите размер общей папки. Емкость многоуровневой общей папки должна составлять от 500 ГБ до 20 ТБ. Для локально закрепленных общих папок это значение должно быть в диапазоне от 50 ГБ до 2 ТБ.

    5. В поле **Set default full permissions to** (Предоставить полные права по умолчанию) предоставьте разрешения пользователю или группе, которые будут обращаться к этой общей папке. Укажите имя пользователя или группы пользователей в формате _john@contoso.com_. Чтобы предоставить права администратора для доступа к этим общим папкам, рекомендуем использовать группы пользователей (вместо одного пользователя). Назначив разрешения здесь, вы сможете изменять их в проводнике.
3. После завершения настройки общей папки нажмите кнопку **Создать**. Будет создана общая папка с указанными настройками. Вы увидите соответствующее уведомление. По умолчанию для нее включено резервное копирование.
4. Чтобы убедиться, что общая папка создана, перейдите в колонку **Общие папки**. Общая папка должна отображаться в списке.
   
    ![Общая папка успешно создана](./media/storsimple-virtual-array-manage-shares/share-success.png)

## <a name="modify-a-share"></a>Изменение общей папки

Вы можете изменить только описание общей папки. Другие свойства общей папки нельзя изменить после ее создания.

#### <a name="to-modify-a-share"></a>Изменение общей папки

1. В разделе **Общие папки** в колонке сводки службы StorSimple выберите виртуальный массив, в котором находится нужная вам общая папка.
2. **Выберите** общую папку, чтобы просмотреть текущее описание и изменить его.
3. Чтобы сохранить изменения, щелкните **Сохранить** на панели команд. Указанные параметры будут применены, и появится соответствующее уведомление.
   
    ![ Изменение общей папки](./media/storsimple-virtual-array-manage-shares/share-edit.png)

## <a name="take-a-share-offline"></a>Отключение общей папки

Если общую папку необходимо изменить или удалить, ее нужно отключить. Отключенная общая папка недоступна для чтения и записи. Отключать общую папку следует на узле и на устройстве.

#### <a name="to-take-a-share-offline"></a>Отключение общей папки

1. Перед отключением общей папки убедитесь, что она не используется.
2. Чтобы отключить общую папку, выполните следующие действия.
   
    1. В разделе **Общие папки** в колонке сводки службы StorSimple выберите виртуальный массив, в котором находится нужная общая папка.

    2. **Выберите** общую папку и нажмите кнопку **…** (или щелкните правой кнопкой мыши в этой строке). Затем в контекстном меню выберите **Отключить**.
     
        ![Отключенная общая папка](./media/storsimple-virtual-array-manage-shares/shares-offline.png)

    3. Просмотрите сведения в колонке **Отключение** и подтвердите операцию. Щелкните **Отключить**, чтобы перевести общую папку в автономный режим. Появится уведомление о выполняющейся операции.

    4. Чтобы убедиться, что общая папка отключена, перейдите в колонку **Общие папки**. Состояние общей папки должно указывать, что она переведена в автономный режим.

## <a name="delete-a-share"></a>Удаление общей папки

> [!IMPORTANT]
> Общую папку можно удалить только в том случае, если она отключена.


Вот как можно удалить общую папку.

#### <a name="to-delete-a-share"></a>Удаление общей папки

1. В разделе **Общие папки** в колонке сводки службы StorSimple выберите виртуальный массив, в котором находится нужная общая папка.
2. **Выберите** общую папку и нажмите кнопку **…** (или щелкните правой кнопкой мыши в этой строке). Затем в контекстном меню выберите **Удалить**.
   
    ![Удаление папки](./media/storsimple-virtual-array-manage-shares/share-delete.png)
3. Проверьте состояние общей папки, которую требуется удалить. Если общая папка, которую вы хотите удалить, не находится в автономном режиме, сначала отключите ее. Следуйте указаниям по [отключению общей папки](#take-a-share-offline).
4. При появлении запроса на подтверждение в колонке **Удалить** подтвердите операцию и щелкните **Удалить**. Общая папка будет удалена, и в колонке **Общие папки** появится обновленный список общих папок в виртуальном массиве.
