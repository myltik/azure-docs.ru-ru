## <a name="may-07-2018"></a>07 мая 2018 г.

### <a name="net-sdk"></a>Пакет SDK для .NET

Пакет SDK для .NET предоставляет следующие возможности:

1. **Преобразования** и **задания**, позволяющие кодировать или анализировать мультимедийное содержимое. Примеры см. в руководстве по [потоковой передаче](stream-files-tutorial-with-api.md) и [анализе видео](analyze-videos-tutorial-with-api.md).
2. **Указатели потоковой передачи**, отвечающие за публикацию и потоковую передачу содержимого на устройства пользователей.
3. **Политики потоковой передачи** и **ключа содержимого**, отвечающие за настройку доставки ключей и защиту содержимого (DRM) при его доставке.
4. **События прямой трансляции** и **выходные данные запущенной прямой трансляции**, отвечающие за прием и потоковую трансляцию содержимого.
5. **Ресурсы**, отвечающие за хранение и публикацию содержимого в службу хранилища Azure. 
6. **Конечные точки потоковой передачи**, позволяющие настраивать и масштабировать динамические упаковки, шифрование и потоковую передачу мультимедийного содержимого в реальном времени и по запросу.

### <a name="known-issues"></a>Известные проблемы

Известная проблема:

При отправке задания через URL-адрес HTTPS (JobInputHttp), указывающий на исходное содержимое, убедитесь, что HTTP-сервер поддерживает запрос HEAD. В противном случае задание будет отклонено.
