|[Расширенное обнаружение угроз](azure-threat-detection.md)|Подробное описание подходов Azure к оценке, диагностике и анализу уязвимостей. |[Сетевая безопасность Azure](azure-network-security.md)|Информация о широком диапазоне средств управления сетями, которые можно настроить для повышения безопасности развернутых в Azure решений. Основное внимание уделяется элементам управления безопасности сетей, доступным для клиента.|
|[Операционная безопасность Azure](azure-operational-security.md)|Подробный обзор доступных в Azure технологий и служб безопасности, с которыми клиент непосредственно взаимодействует.|
|[Технические возможности обеспечения безопасности Azure](azure-security-technical-capabilities.md)|Описание функций и компонентов безопасности, предназначенных для службы хранилища Azure, баз данных SQL Azure, модели виртуальных машин Azure, а также вспомогательных средств и инфраструктуры.|

Описание защиты для плоскости управления и плоскости данных, шифрования неактивных и передаваемых данных, а также аналитики хранилища.|

Этот документ будет особенно полезен тем организациям, которые рассматривают применимость облачных вычислений для будущих задач или стремятся оптимизировать процессы управления данными для уже используемых облачных служб.|
|[Управление в Azure](governance-in-azure.md)|Описание функций безопасности и управления, встроенных в Azure. |[Изоляция в общедоступном облаке Azure](azure-isolation.md)|Описание методов, используемых в Azure для изоляции от злоумышленников и других пользователей. Этот документ служит руководством для разработки облачных решений и включает несколько вариантов изоляции. В первую очередь рассматриваются доступные клиентам средства управления безопасностью, но почти не затрагиваются соглашения об уровне обслуживания, модели ценообразования и практические рекомендации для DevOps.|
|[Overview of Microsoft Azure compliance](https://gallery.technet.microsoft.com/Overview-of-Azure-c1be3942) (Общие сведения о соответствии Azure)|В этом загружаемом документе описываются предложения Azure по соответствию, в том числе официальные процедуры сертификаты, аттестации, проверки, авторизации и оценки, проведенные независимыми аудиторскими фирмами, а также поправки к контрактам, самостоятельные оценки и руководства для клиентов, предоставляемые корпорацией Майкрософт. <br/> В описании каждого предложения отдельно указано, к каким клиентским службам Azure относится эта оценка, а также предоставлены ссылки на загружаемые материалы, помогающие пользователям выполнять обязательства по соответствию.|
|[Управление безопасностью в Azure](azure-security-management.md)|Обсуждение проблем, возникающих при удаленном доступе к ресурсам Azure. Для облачных систем администрирование и безопасность удаленного доступа имеют первостепенное значение. Здесь описаны общие рекомендации, конфигурации клиентов, практические указания, принципы и процедуры операционной работы. |
