# <a name="pci-dss-requirements---high-level-overview"></a>Требования PCI DSS — общий обзор

Стандарт безопасности данных индустрии платежных карт (PCI DSS) был разработан для обеспечения и оптимизации безопасности данных владельцев карт, а также для глобального внедрения согласованных мер по защите целостности данных. PCI DSS предоставляет базовые технические и эксплуатационные требования, предназначенные для защиты данных счетов. Стандарты PCI DSS применяются ко всем сущностям, участвующим в обработке карт оплаты, включая продавцов, обработчиков, получателей, издателей и поставщиков услуг. Они также применяются к остальным сущностям, хранящим, обрабатывающим или передающим данные владельцев карт (CHD) и/или конфиденциальные данные аутентификации (SAD). Ниже приведен общий обзор 12 требований PCI DSS.

> [!NOTE]
> Эти требования определены [Советом по стандартам безопасности индустрии платежных карт (Payment Card Industry Security Standards Council, PCI SSC)](https://www.pcisecuritystandards.org/pci_security/) в [стандарте PCI DSS версии 3.2](https://www.pcisecuritystandards.org/document_library?category=pcidss&document=pci_dss).
