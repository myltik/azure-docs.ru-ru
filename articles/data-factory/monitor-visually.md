# <a name="visually-monitor-azure-data-factories"></a>Визуальный мониторинг фабрик данных Azure
Фабрика данных Azure — это облачная служба интеграции данных, которая позволяет создавать управляемые данными рабочие процессы в облаке для оркестрации и автоматизации перемещения и преобразования данных. С помощью фабрики данных Azure можно создавать и включать в расписание управляемые данными рабочие процессы (конвейеры), которые могут принимать данные из разнородных хранилищ данных, обрабатывать и преобразовывать эти данные с помощью служб вычислений (например, Azure HDInsight Hadoop, Spark, Azure Data Lake Analytics и машинного обучения Azure), а также публиковать выходные данные в хранилища данных (например, хранилище данных SQL Azure) для использования приложениями бизнес-аналитики.
В этом кратком руководстве вы узнаете, как визуально отслеживать конвейеры фабрики данных версии 2 без применения программного кода.
Если у вас еще нет подписки Azure, создайте [бесплатную](https://azure.microsoft.com/free/) учетную запись Azure, прежде чем начинать работу.

> [!NOTE]
> Эта статья относится к версии 2 фабрики данных, которая в настоящее время доступна в предварительной версии. Если вы используете общедоступную версию 1 службы фабрики данных, прочитайте статью [Мониторинг конвейеров фабрики данных Azure и управление ими с помощью приложения для мониторинга и управления](v1/data-factory-monitor-manage-app.md).

## <a name="monitor-data-factory-v2-pipelines"></a>Мониторинг конвейеров в фабрике данных версии 2

1. Запустите веб-браузер **Microsoft Edge** или **Google Chrome**. Сейчас только эти браузеры поддерживают пользовательский интерфейс фабрики данных.
2. Войдите на [портал Azure](https://portal.azure.com/).
3. Перейдите к колонке созданной фабрики данных и щелкните плитку "Мониторинг и управление". Откроется визуальный интерфейс мониторинга для ADF версии 2.

## <a name="list-view-monitoring"></a>Мониторинг в представлении списка

Выполнения конвейера и действий можно отслеживать в простом интерфейсе с представлением списка. Все выполнения отображаются по времени часового пояса, который настроен в браузере. Если вы измените локальные настройки часового пояса, все поля даты и времени автоматически переключатся на новый часовой пояс.  

#### <a name="monitoring-pipeline-runs"></a>Мониторинг выполнений конвейера
Представление списка демонстрирует все выполнения для каждого конвейера из фабрики данных версии 2. В представление включаются следующие столбцы.

| **Имя столбца** | **Описание** |
| --- | --- |
| Pipeline Name | Имя конвейера. |
| Действия | Здесь доступно одно действие для просмотра выполнений действий. |
| Run Start | Дата и время начала выполнения конвейера (ММ/ДД/ГГГГ ЧЧ:ММ:СС AM/PM) |
| Duration | Длительность выполнения (ЧЧ:ММ:СС) |
| Triggered By | Срабатывание по ручному (Manual) или запланированному (Schedule) триггеру. |
| Status | Сбой (Failed), Успешно (Succeeded), Выполняется (In Progress) |
| Параметры | Параметры запуска конвейера (пары "имя — значение") |
| Ошибка | Ошибки выполнения конвейера (если есть) |
| Run ID | Идентификатор выполнения конвейера. |

![Мониторинг выполнений конвейера](media/monitor-visually/pipeline-runs.png)

#### <a name="monitoring-activity-runs"></a>Мониторинг выполнений действий
Представление списка отображает все выполнения действий, относящиеся к каждому выполнению конвейеров. Щелкните значок **Выполнения действий** под столбцом **Действия**, чтобы увидеть все выполнения действий для каждого выполнения конвейера. В представление включаются следующие столбцы.

| **Имя столбца** | **Описание** |
| --- | --- |
| Activity Name | Имя действия в конвейере. |
| Activity Type | Тип действия, например копирование (Copy), HDInsightSpark, HDInsightHive и т. д. |
| Run Start | Дата и время начала выполнения действия (ММ/ДД/ГГГГ ЧЧ:ММ:СС AM/PM) |
| Duration | Длительность выполнения (ЧЧ:ММ:СС) |
| Status | Сбой (Failed), Успешно (Succeeded), Выполняется (In Progress) |
| Входные данные | Массив JSON, описывающий входные данные действия |
| Выходные данные | Массив JSON, описывающий выходные данные действия |
| Ошибка | Ошибки выполнения действия (если есть) |

![Мониторинг выполнений действий](media/monitor-visually/activity-runs.png)

> [!IMPORTANT]
> Щелкните значок **Обновить** вверху, чтобы обновить список выполнений конвейеров и (или) действий. Автоматическое обновление в настоящее время не поддерживается.
>

![Обновить](media/monitor-visually/refresh.png)

## <a name="features"></a>Функции

#### <a name="rich-ordering-and-filtering"></a>Широкие возможности поиска и фильтрации

Список выполнений конвейеров можно упорядочить по времени запуска (по возрастанию или по убыванию), а также отфильтровать по значениям следующих столбцов.

| **Имя столбца** | **Описание** |
| --- | --- |
| Pipeline Name | Имя конвейера. Предусмотрены быстрые фильтры, например "за последние 24 часа", "за последнюю неделю", "за последние 30 дней", также здесь можно выбрать произвольные дату и время. |
| Run Start | Дата и время начала выполнения конвейера |
| Run Status | Выбор выполнений по определенному состоянию: Сбой (Failed), Успешно (Succeeded), Выполняется (In Progress) |

![Фильтр](media/monitor-visually/filter.png)

#### <a name="addremove-columns-to-list-view"></a>Добавление и удаление столбцов в представлении списка
Щелкните правой кнопкой мыши заголовок представления списка и выберите столбцы, которые вы хотите отображать в этом представлении

![Столбцы](media/monitor-visually/columns.png)

#### <a name="reorder-column-widths-in-list-view"></a>Изменение ширины столбцов в представлении списка
Чтобы увеличить или уменьшить ширину столбца в представлении списка, просто наведите указатель мыши на заголовок столбца

#### <a name="select-data-factory"></a>Выбор фабрики данных
Наведите указатель мыши на значок "Фабрика данных" в верхнем левом углу. Щелкните значок "Стрелка", чтобы просмотреть список подписок Azure и фабрик данных, которые доступны для мониторинга.

![Выбор фабрики данных](media/monitor-visually/select-datafactory.png)

#### <a name="guided-tours"></a>Интерактивные руководства
Щелкните значок "Сведения" в нижнем левом углу и выберите действие "Интерактивные руководства", чтобы получить пошаговые инструкции по мониторингу выполнений конвейеров и действий.

![Интерактивные руководства](media/monitor-visually/guided-tours.png)

#### <a name="feedback"></a>Отзыв
Щелкните значок "Отзыв", чтобы поделиться мнением об используемых функциях или сообщить о проблемах, с которыми вы столкнулись.

![Отзыв](media/monitor-visually/feedback.png)
