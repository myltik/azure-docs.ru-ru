# <a name="review-your-access"></a>Проверка доступа

Azure Active Directory (Azure AD) позволяет предприятиям упростить управление доступом для приложений и участников групп в Azure AD, а также других служб Microsoft Online Services с помощью функции проверки доступа. Вы могли получить от корпорации Майкрософт электронное сообщение, предлагающее вам проверить доступ участников группы или пользователей, имеющих доступ к приложению. 

## <a name="open-an-access-review"></a>Открытие проверки доступа

Чтобы просмотреть ожидающие проверки доступа, щелкните ссылку в сообщении. Если вам не пришло сообщение, то вы можете найти проверки доступа, выполнив следующие действия.

1. Войдите на [панель доступа Azure AD](https://myapps.microsoft.com).

2. Щелкните значок пользователя в правом верхнем углу страницы, где отображается ваше имя и название организации по умолчанию. Если в списке содержится несколько организаций, выберите ту организацию, которая запросила проверку доступа.

3. Если в правой части страницы имеется элемент **Проверки доступа**, щелкните его. Если такой элемент не отображается, для этой организации нет ожидающих проверок доступа и в данный момент не требуется никаких действий.

## <a name="fill-out-an-access-review"></a>Заполнение проверки доступа

Выбрав проверку доступа из списка, можно просмотреть сведения о вашем доступе. Щелкните запись и выберите необходимое действие: утвердить или отклонить необходимость получения доступа.

Рецензент может потребовать, чтобы вы указали обоснование для продолжения предоставления доступа.

## <a name="next-steps"></a>Дополнительная информация

Запрещенный доступ не удаляется мгновенно. Если вы хотите изменить свой ответ и утвердить доступ, сбросьте ответ и выберите новый ответ. Это можно сделать до завершения проверки доступа.
