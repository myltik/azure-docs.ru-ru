# <a name="create-list-or-delete-a-user-assigned-managed-service-identity-msi-using-the-azure-portal"></a>Создание, вывод списка и удаление пользовательских управляемых удостоверений службы (MSI) с помощью портала Azure

[!INCLUDE[preview-notice](~/includes/active-directory-msi-preview-notice-ua.md)]

Управляемое удостоверение службы предоставляет службы Azure с управляемыми удостоверениями в Azure Active Directory. Это удостоверение можно использовать для аутентификации в службах, которые поддерживают аутентификацию Azure AD, без учетных данных в коде. 

В настоящее время с помощью портала Azure невозможно создать или удалить пользовательское управляемое удостоверения либо получить их список.  Загляните сюда позже, чтобы проверить наличие новой информации.
