# <a name="understand-free-service-to-meter-mapping"></a>Сопоставление единиц измерения бесплатных служб

Использование каждой службы Azure включает показатели, которые использует система выставления счетов Azure, чтобы взимать плату с пользователей за использование служб. Чтобы лучше ознакомиться с использованием бесплатных служб, рассмотрим сопоставление единиц измерения для этих служб. Чтобы узнать, как создать бесплатные службы, ознакомьтесь со статьей о [создании бесплатных служб с помощью бесплатной учетной записи Azure](billing-create-free-services-included-free-account.md).
