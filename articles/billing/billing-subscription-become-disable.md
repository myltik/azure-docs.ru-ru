# <a name="why-is-my-azure-subscription-disabled-and-how-do-i-reactivate-it"></a>Почему подписка Azure отключена и как активировать ее повторно?
Подписка Azure может быть отключена, так как достигнута предельная сумма расходов, есть просроченный платеж, исчерпан лимит кредитной карты или администратор учетной записи отменил подписку. Выберите подходящую ситуацию и выполните действия, описанные в этой статье, чтобы активировать подписку повторно.

## <a name="you-reached-your-spending-limit"></a>Достигнута предельная сумма расходов
При достижении максимального ежемесячного ограничения на использование для учетной записи служба будет отключена до окончания месяца выставления счета. Это делается, чтобы предотвратить тарификацию за случайное использование сверх ограничений в рамках подписки. Вы можете удалить предельную сумму расходов и [настроить предупреждения о выставлении счета для подписок Microsoft Azure](billing-set-up-alerts.md), чтобы отслеживать и контролировать действия по выставлению счетов для учетных записей Azure.

Удаление предельной суммы расходов

1. Войдите в [центр учетных записей Azure](https://account.windowsazure.com/Home/Index).
2. Щелкните **Подписки**.
3. Выберите подписку.
4. Щелкните **Subscription reached the Spending Limit and has been disabled to prevent charges** (Достигнута предельная сумма расходов по подписке, подписка отключена для предотвращения перерасхода средств).

    Если этот вариант не отображается, в области **Состояние** подписки щелкните **Снять предельную сумму расходов**.
5. Выберите один из следующих вариантов:

| Параметр | Результат |
| --- | --- |
| Удалить предельную сумму расходов на неопределенный срок |Удаляет предельную сумму расходов, пока она снова не будет включена. |
| Удалить предельную сумму расходов для текущего расчетного периода |Удаляет предельную сумму расходов. При этом она перезапускается со следующего расчетного периода. |

Если у вас есть бесплатная пробная подписка, после удаления предельной суммы расходов оплата за эту подписку в конце срока будет выставлена по мере ее использования.

## <a name="your-bill-is-past-due"></a>Счет просрочен
Полностью оплатить задолженность. См. статью [Почему вы получили уведомление о том, что имеется задолженность по вашей подписке Azure?](billing-azure-subscription-past-due-balance.md#how-to-resolve-the-issue)

## <a name="the-bill-exceeds-your-credit-card-limit"></a>Сумма счета превышает баланс кредитной карты
Для решения этой проблемы [выберите другую кредитную карту](billing-how-to-change-credit-card.md). Если вы являетесь представителем компании, вы можете [выбрать оплату по выставленному счету](https://azure.microsoft.com/pricing/invoicing/).

## <a name="the-subscription-was-canceled-by-the-account-administrator"></a>Подписку отменил администратор учетной записи
Если вы являетесь администратором учетной записи, то вы можете повторно активировать подписку с оплатой по мере пользования, выполнив следующие действия:

1. Войдите в [центр учетных записей Azure](https://account.windowsazure.com/Home/Index).
2. Выберите отмененную подписку.
3. Щелкните **Повторная активация**.

![Снимок экрана, на котором показаны ссылки для повторной активации в области справа.](./media/billing-how-to-cancel-azure-subscription/reactivate-sub.png)

Для повторной активации подписок других типов [обратитесь в службу поддержки](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade).

## <a name="need-help-contact-support"></a>Требуется помощь? Обратитесь в службу поддержки.
Если вам все еще нужна помощь, [обратитесь в службу поддержки](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade), которая поможет быстро устранить проблему.
