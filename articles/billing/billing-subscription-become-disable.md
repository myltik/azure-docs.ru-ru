# <a name="why-is-my-azure-subscription-disabled-and-how-do-i-reactivate-it"></a>Почему подписка Azure отключена и как активировать ее повторно?

Подписка Azure может быть отключена, так как срок действия денег на счете истек, достигнута предельная сумма расходов, есть просроченный платеж, исчерпан лимит кредитной карты или администратор учетной записи отменил подписку. Выберите подходящую ситуацию и выполните действия, описанные в этой статье, чтобы активировать подписку повторно.

## <a name="your-credit-is-expired"></a>Истек срок действия денег на счете

При регистрации бесплатной учетной записи Azure вы получите бесплатную пробную версию подписки, которая предоставляет 200 долл. на счете в Azure на 30 дней и 12 месяцев бесплатных служб. Через 30 дней Azure отключит вашу подписку. Это делается, чтобы предотвратить тарификацию за случайное использование денег на счете и бесплатных служб сверх ограничений в рамках подписки. Чтобы продолжить использование служб Azure, необходимо [обновить подписку до подписки с оплатой по мере использования](billing-upgrade-azure-subscription.md). После обновления ваша подписка по-прежнему имеет доступ к бесплатным службам в течение 12 месяцев. Плата взимается только за использование платных служб и ресурсов.

## <a name="you-reached-your-spending-limit"></a>Достигнута предельная сумма расходов

В подписках Azure с деньгами на счете, таких как бесплатная пробная версия и Visual Studio Enterprise, есть предельные суммы расходов на них. Это означает, что использовать службы можно только в пределах предоставленных денег на счете. По достижении предельной суммы расходов Azure отключает подписку до истечения периода выставления счетов. Это делается, чтобы предотвратить тарификацию за случайное использование денег на счете сверх ограничений в рамках подписки. Дополнительные сведения см. в разделе [Снятие ограничения на предельную сумму расходов](billing-spending-limit.md#remove).

> [!NOTE] 
> Если у вас есть бесплатная пробная подписка, после отключения предельной суммы расходов оплата за эту подписку в конце срока ее действия будет выставлена по мере использования. После создания подписки оставшиеся деньги на счете хранятся в течение полных 30 дней. Кроме того, вам предоставляется доступ к бесплатным службам в течение 12 месяцев.

Сведения об отслеживании действий по выставлению счетов для Azure и управлении этими действиями см. в статьях [Предотвращение непредвиденных расходов с помощью функции выставления счетов и управления затратами в Azure](billing-getting-started.md) и [Настройка оповещений о выставлении счетов или предоставлении кредитов для подписок Microsoft Azure](billing-set-up-alerts.md).


## <a name="your-bill-is-past-due"></a>Счет просрочен

Сведения об устранении этой проблемы см. в статье [Решение проблемы просроченных платежей по подписке Azure после получения электронного сообщения от Azure](billing-azure-subscription-past-due-balance.md).

## <a name="the-bill-exceeds-your-credit-card-limit"></a>Сумма счета превышает баланс кредитной карты

Для решения этой проблемы [выберите другую кредитную карту](billing-how-to-change-credit-card.md). Если вы являетесь представителем компании, вы можете [выбрать оплату по выставленному счету](billing-how-to-pay-by-invoice.md).

## <a name="the-subscription-was-accidentally-canceled-and-you-want-to-reactivate"></a>Подписка была случайно отменена, и вы хотите повторно активировать ее

Если вы являетесь администратором учетной записи и случайно отменили подписку с оплатой по мере использования, ее можно повторно активировать в Центре управления учетной записью.

1. Войдите в [Центр управления учетной записью](https://account.windowsazure.com/Subscriptions).
1. Выберите отмененную подписку.
1. Щелкните **Повторная активация**.

    ![Снимок экрана, на котором показаны ссылки для повторной активации в области справа.](./media/billing-how-to-cancel-azure-subscription/reactivate-sub.png)

Для повторной активации подписок других типов [обратитесь в службу поддержки](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade).
