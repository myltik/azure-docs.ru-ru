# <a name="azure-network-watcher-documentation"></a>Документация по Наблюдателю за сетями Azure

Сведения об использовании Наблюдателя за сетями — службы, обеспечивающей мониторинг и диагностику условий на уровне сети. В руководствах, справочниках по API и другой документации показано, как получать ценные сведения о сети Azure с помощью таких средств, как захват пакетов, топология и многое другое.
