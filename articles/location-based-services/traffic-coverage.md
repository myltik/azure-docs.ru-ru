# <a name="azure-location-based-services---traffic-coverage"></a>Покрытие трафика в Azure Location Based Services

Служба Azure Location Based Services (LBS) предоставляет множество сведений о трафике в виде потока и инцидентов, которые можно использовать в качестве наложений поверх Azure LBS Map Control или совместно со Службой построения маршрутов Azure LBS. В следующей таблице представлена информация о покрытии для Azure LBS Traffic, чтобы пользователи имели представление о том, где они смогут (и не смогут) получить информацию о трафике.
