# <a name="web-app-on-linux-documentation"></a>Документация по веб-приложению на платформе Linux
В настоящее время веб-приложение на платформе Linux доступно в виде общедоступной предварительной версии. Оно позволяет клиентам выполнять веб-приложения изначально на платформе Linux. Это обеспечивает лучшую совместимость определенных типов приложений и упрощает процесс переноса готовых веб-приложений, размещенных на платформе Linux, в службу приложений Azure.

Клиенты могут создавать веб-приложения на платформе Linux только в рамках плана службы приложений с поддержкой Linux. В рамках этого плана можно размещать только веб-приложения на базе Linux. 

> Цена на общедоступную предварительную версию веб-приложения на платформе Linux на 50 % ниже обычной. Она рассчитывается на основе количества часов вычислений, потребленных в рамках используемого плана службы приложений.
> 
> На этапе общедоступной предварительной версии соглашение об уровне обслуживания не предоставляется.
